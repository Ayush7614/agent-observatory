Built for real agent workflows

Not another cloud SaaS. A command center that stays on your machine.

Live activity

Watch Read, Edit, Bash, and Grep stream in real time via SSE and Claude Code hooks.

Session search

Full-text search across every indexed session. Find that bug fix from last week in seconds.

Export & recover

One-click Markdown export or ao recover — never lose context after a crash.

Ollama friendly

Local models show $0 / Free · local. Cloud models show usage estimates.

Auto-snapshots

Active sessions backed up every 5 minutes to ~/.agent-observatory/snapshots/.

Zero telemetry

No analytics, no phone-home. SQLite store under ~/.agent-observatory/.

Install in 2 minutes

Requires Node.js 22+. Works on macOS and Linux.

git clone https://github.com/Ayush7614/agent-observatory.git
cd agent-observatory
npm install
npm run build:dashboard
npm run start

# Optional: real-time hooks
npm run install-hooks
ao doctor

Open http://127.0.0.1:7420 — the dashboard runs locally, not in the cloud.
